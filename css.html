Frågor

  1. Vad är en regel i CSS och vad består den av?
  2. Vad är skillnaden mellan id och class? När bör man använda id respektive class?

Svar

  1. struktur-regeln i css är att en selector inehåller en egenskap och ett värde. Selectorn är namnet på våran class/pseudo-class/html element/id/attribut som vi vill designa. Man kan även tilldela selectorn med yttligare information som till exempel :hover, :active mm. En egenskap är en utav de existerande egenskapeerna som finns tex; fontsize, display, width, height. Värdet är det som du vill ge elementet tex; 18px, flex, 20%, 100rem. Ifall selectorn är ett html element så behövs inget tecken men en class behöver en punkt innan namnet och id behöver ett #. För att använda flera element som en selector så skrivs de tillsammans med ett kommatecken emellan dem.

  2. En class är ämnad för att designa ett speciellt element och kan repiteras, medans ett id används endast en gång och ska vara unikt. Detta är för att ett id kan, förutom att användas i css så som en class, även används i funktioner i bland annat jacascript. Om ett id används som en class, dvs repiteras på flera olika platser, så finns det en risk att det skapar problem med olika funktioner längre fram. Därför ska ett id användas när du vill referera det elementet längre fram i tex en function med; document.getElementById('id-name'). Med andra ord, en class används för att referera till en eller flera element, kan repiteras och används til mesta dels när man vill designa dessa element i css. Ett id kan användas på samma sätt som en class i css men får inte repiteras. Ett id används till största dels när man vill referensera ett element i en function.
